Project Title

Tic Tac Toe Web Application
-----------------------------------------------------------------------------------------------------------
Description

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. 
The application allows two players to play the game in turns on a 3×3 grid. 
It detects winning combinations, identifies draw situations, and provides a reset option to start a new game.
The project was created to practice DOM manipulation, event handling, and basic game logic in JavaScript.
--------------------------------------------------------------------------------------------------------------
Technologies Used

HTML   CSS   JavaScript
--------------------------------------------------------------------------------------------------------------
Learning Outcomes

This project helped me understand:

-How to manipulate the DOM using JavaScript
-Handling user events like clicks
-Implementing logic for game rules
-Structuring a small web project
